# Microsoft Security Ecosystem Overview

## Components
- Microsoft Sentinel (SIEM)
- Defender for Endpoint (EDR)
- Defender for Identity
- Defender for Cloud

## Alert Flow
1. Endpoint generates telemetry
2. Defender analyzes behavior
3. Alert created
4. Sentinel ingests alert
5. Incident correlation happens
